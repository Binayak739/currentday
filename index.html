How to create a Gmail account?

To create a Gmail account, go to the Gmail sign-up page, enter your name, choose a username, create a password, and provide your birthdate. Optionally, add a phone number and recovery email. Agree to the terms, complete any verification steps, and you’re done!

How to create a GitHub account?

To create a GitHub account, visit GitHub’s sign-up page. Enter your username, email, and password. Verify your email address. Choose your plan (free or paid). Follow the on-screen instructions to complete setup. You’re ready to start hosting and sharing your code on GitHub!

How to connect VS Code editor to GitHub account?

To connect VS Code to your GitHub account, follow these steps:
Install Git: Download and install Git from git-scm.com.
Install GitHub Extension: Open VS Code, go to the Extensions view (click the square icon or press Ctrl+Shift+X), and search for "GitHub." Install the “GitHub Pull Requests and Issues” extension.
Sign In: Click the "Sign in to GitHub" prompt in the bottom-right corner or go to the Source Control view (click the branch icon or press Ctrl+Shift+G) and sign in.
Clone or Initialize: Clone a repository or initialize a new one by using the commands in the Source Control view.
You’re all set to start coding and syncing with GitHub in VS Code!

How to clone/pull repository from GitHub account?

To clone a GitHub repository:
Copy the repository's URL from GitHub.
Open your terminal or Git Bash.
Enter git clone [URL] and press Enter.
To pull updates:
Navigate to the repository's folder.
Enter git pull to fetch and merge updates.
That's it!

How to push repository folder to GitHub?

To push a folder to GitHub:
Navigate to the folder in the terminal.
Initialize with git init.
Add files with git add ..
Commit changes with git commit -m "Initial commit".
Link the remote repo with git remote add origin [URL].
Push with git push -u origin main.

How to publish the repository as a live site?

To publish a GitHub repository as a live site using GitHub Pages:
Go to Your Repository: Open your repository on GitHub.
Navigate to Settings: Click the "Settings" tab.
Access Pages Settings: Scroll down to the "Pages" section in the left sidebar.
Choose a Source:
Select the branch you want to publish (usually main or gh-pages).
Optionally, choose the folder (typically /root or /docs).
Save and Wait: Click "Save." GitHub will process the site.
Access Your Site: After a few minutes, your live site will be available at https://your-username.github.io/repository-name.
Your site is now live!
